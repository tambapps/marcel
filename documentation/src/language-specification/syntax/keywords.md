# Keywords

Marcel has the following keywords, which you cannot use as variable/function names


- int
- long
- short
- float
- double
- bool
- byte
- void
- char
- fun
- return
- true
- false
- new
- import
- as
- inline
- static
- for
- in
- if
- else
- null
- break
- def
- class
- extension
- package
- extends
- implements
- final
- switch
- when
- this
- super
- dumbbell
- try
- catch
- finally
- instanceof
